❓ FAQ - Domande Frequenti

L'app è gratuita?

Sì, UniStudy Italia è completamente gratuita e senza pubblicità.

Perché devo dare il permesso alla posizione?

Il permesso di localizzazione serve per mostrarti sulla mappa e per ordinare le aule studio in base alla distanza. Non raccogliamo né inviamo i tuoi dati di posizione a server esterni.

Posso usare l'app offline?

Sì, i dati delle aule sono memorizzati localmente. Tuttavia, le mappe richiedono una connessione internet per caricarsi.
