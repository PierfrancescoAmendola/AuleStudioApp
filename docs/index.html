🌟 Funzionalità Principali

🏫 Multi-Ateneo

Supporto per oltre 100 atenei: PoliMi, La Sapienza, Federico II, UniTO, UniBa e molti altri, inclusi Conservatori e Accademie (AFAM).

🔍 Filtri Intelligenti

Filtra per Regione, Edificio o Polo Universitario. Trova subito le aule aperte o quelle all'aperto grazie alla SmartWeather integration.

📍 Navigazione

Calcolo della distanza in tempo reale e indicazioni precise (Bus, Metro, Piedi) integrate con le tue mappe native.

🎨 Theming Dinamico

L'app cambia colore istantaneamente in base alla tua università. Un'esperienza personalizzata e identitaria.

🛠 Stack Tecnologico

UniStudy è costruita con tecnologie all'avanguardia per garantire stabilità e fluidità a 60 FPS:

React Native
Expo SDK
TypeScript
React Navigation
Native Maps
AsyncStorage

🧪 Qualità e Testing

La stabilità è la nostra priorità. Utilizziamo un approccio di testing rigoroso per garantire un'esperienza impeccabile su ogni dispositivo:

✅ Unit & Component Testing

Test automatici con Jest per validare logiche di calcolo orari e filtri complessi.

🤖 Android QA Specifico

Gestione avanzata del rendering nativo per evitare "Ghost Pins" nelle mappe e bug grafici tipici del motore Android.

🛡 Error Handling

Flussi di fallback eleganti per la gestione dei permessi GPS e della connettività offline.
